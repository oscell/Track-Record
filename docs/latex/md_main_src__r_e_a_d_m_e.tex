Track-\/\+Record is a face tracking system built on the Raspberry Pi 3B platform, using a Pi Camera Module and servo motors for real-\/time face detection and tracking. A G\+UI is also implemented which comunicates via a server sending commands to the raspi.

The full documentation can be found \href{https://tduggan63.github.io/Track-Record/index.html}{\texttt{ here}}.
